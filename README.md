# Conway's Game of Life Simulation
A high performance terminal-based simulation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life). 

## Tech Stack: 
![C++](https://img.shields.io/badge/c++-%2300599C.svg?style=for-the-badge&logo=c%2B%2B&logoColor=white)

## Prerequisits: 
To run this game, you'll need a Windows environment with a C++ compiler installed
* [MinGW-w64](https://www.msys2.org/) (for `g++` and `mingw32-make`)
